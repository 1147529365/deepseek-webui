# DeepSeek WebUI

DeepSeek WebUI 是一个基于 DeepSeek 大语言模型的现代化 Web 交互界面。

## 主要功能

### 对话功能
- 支持多轮对话
- 支持代码高亮显示
- 支持 Markdown 渲染
- 支持数学公式渲染
- 支持多会话管理
- 支持对话历史导出/导入

### 模型控制
- 支持温度（temperature）调节
- 支持采样策略选择（top_p, top_k）
- 支持最大输出长度控制
- 支持系统提示词（System Prompt）自定义
- 支持多个模型切换（7B, 67B）

### 高级功能
- 代码解释器集成
- 文件上传和处理
- API 密钥管理
- 对话模板管理
- 提示词（Prompt）模板库

## 技术栈

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Zustand (状态管理)
- localStorage (本地数据存储)
